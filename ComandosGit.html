Comandos Básicos de Git

Lista rápida de comandos essenciais para começar a usar o Git:

  • git init – Inicializa um novo repositório Git.
  • git clone <url> – Clona um repositório remoto para sua máquina.
  • git status – Mostra o estado atual dos arquivos no repositório.
  • git add <arquivo> – Adiciona arquivos ao índice (staging area).
  • git commit -m "mensagem" – Salva as alterações com uma mensagem descritiva.
  • git push – Envia os commits para o repositório remoto.
  • git pull – Atualiza seu repositório local com as mudanças do remoto.
  • git log – Exibe o histórico de commits.
